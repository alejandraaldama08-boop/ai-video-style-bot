============================================
CLIPFORGE - PAQUETE DE RENDER LOCAL
============================================

Este paquete contiene todo lo necesario para renderizar tu edit localmente.

CONTENIDO:
- edit_plan.json: Plan de edición generado por IA
- render.sh: Script de FFmpeg para ejecutar el render
- README.txt: Este archivo

REQUISITOS:
- FFmpeg instalado (https://ffmpeg.org/download.html)
- macOS, Linux o Windows con WSL/Git Bash

INSTRUCCIONES:
1. Crea una carpeta nueva para tu proyecto
2. Copia este paquete a esa carpeta
3. Descarga tus clips originales y colócalos en la carpeta
4. Renombra los clips como: clip_0.mp4, clip_1.mp4, etc.
5. Renombra tu música como: music.mp3
6. Ejecuta el script:
   chmod +x render.sh
   ./render.sh

CONFIGURACIÓN DEL EDIT:
- Resolución: 1080x1920
- Aspecto: 9:16
- FPS: undefined
- Clips: 24 segmentos
- Música: Sí

CLIPS NECESARIOS:
- clip_0.mp4
- clip_1.mp4
- clip_2.mp4
- clip_3.mp4
- clip_4.mp4
- clip_5.mp4
